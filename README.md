# cloxx

__cloxx__ is an interpreter for [Crafting Interpreters'](http://www.craftinginterpreters.com/) [Lox](http://www.craftinginterpreters.com/the-lox-language.html) language written in C++.

## How to build cloxx

```
$ script/configure.sh
$ script/build.sh
```
